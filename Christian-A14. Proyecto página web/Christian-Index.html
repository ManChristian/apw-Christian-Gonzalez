Párrafos y Formato de Texto

Este es un **párrafo** (`p`) introductorio. Utilizamos texto en **negrita (`strong`)** para destacar, y texto en _cursiva (`em`)_ para énfasis. Las etiquetas básicas como negrita (`b`) e cursiva (`i`) aún se admiten. El texto resaltado (`mark`) es clave.

Aquí hay texto más técnico con una nota legal (`small`). También demostramos el subíndice (SO₂) (`sub`) y el superíndice (10⁴) (`sup`). Este texto pertenece a la división con atributo `id`.
